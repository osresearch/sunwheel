Sextant specialized slide rule
Observation
Correcting sextant height measurement:
  • Start with the pointer on the inner and outer at 0
  • Rotate the outer dial to the minutes of the sextant height (black digits).
  • Correct for sextant index error: Point to the index error on the inner dial (red digits), read the corrected height from the outer dial
  • Correct for height of eye, temperature and refraction: Rotate the inner dial so that the height of eye is under the pointer.
  • Rotate the pointer so that it lines up with the current temperature and height of the measurement.
  • Read the refraction and temperature corrected height measurement from the outer ring.
  • Correct for upper/lower limb measurement: For upper limb measurement, rotate the inner to line up the approximate time of year with the pointer.
    For lower, rotate the inner to zero.
    For stars, skip this step.
  • For upper limb measurements, rotate the pointer to the zero of the inner ring. For lower, rotate the pointer to the approximate time of year.
  • Read the semi-diameter corrected height measurement from the outer ring.
Computing latitude (same hemisphere, L > D):
  • L > D, so L=90-(H-D), which means subtracting the sun declination from the sextant height. Rotate the inner dial so that 2.4 in black lines up with the pointer, which will move the noon value higher.
  • d is positive and we are subtracting the declination change. Rotate the pointer to where the 0.8 line intersects the 21:21 with the red numbers.
  • Read the minutes off the black numbers on the outer dial and compute the H-D value as 35° 4.7m - 12° = 23° 4.7' (the minutes of declination have already been handled).
  • Compute the latitude as 90 - 23° 4.7'. 60' - 4.7' can be read from the red numbers on the outer scale, while the sine/cosine scale on the reverse can be used to estimate the degrees 90-23 = 66 point something. The result is 66° 55.3'.
